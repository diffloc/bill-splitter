Check if a number is a palindrome

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
You need to find out if a given number is a palindrome or not. A palindrome number is a number that remains the same when its digits are reversed. Write a program that takes an integer as an input and outputs 'palindrome' if the given integer is a palindrome, else 'not palindrome'.
Sample Input:
12321

Sample Output:
palindrome


Sample Input:
12345

Sample Output:
not palindrome



Memory limit: 256 MB
Time limit: 5 seconds
