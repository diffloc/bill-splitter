Converting strings to triple quoted strings

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
You are provided with a single line quoted string that may contain multiple lines of sentences. Your task is to rewrite the string in a way that it represents the exact same string but is now surrounded by triple quotes '"""' instead of single or double quotes at the start and end. Both the input string and the output string should not contain the escape character '\' (backslash).
Sample Input:
'Hello,\nWorld!'

Sample Output:
"""Hello,\nWorld!"""


Sample Input:
"I'm learning Python.\nIt's fun!"

Sample Output:
"""I'm learning Python.\nIt's fun!"""



Memory limit: 256 MB
Time limit: 5 seconds
