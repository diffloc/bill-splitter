Mess with variables

Wow! This problem is kind of tricky. If you're ready to put your thinking cap on, brace yourself and good luck! Otherwise, you can skip it for now and return any time later.

Look at the variables defined below:

a = 50
b = 5
c = 10

Put the lines in the correct order to reach the following results:

a = 5
b = 50
c = 5

Tip: Are you sure that the value that can be assigned to a particular variable doesn't affect the others?
