Determining if a user-inputted number is even

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a Python program that takes a number (integer or float) as input from the user and returns True if the number is even; otherwise, it returns False. Please consider that if a float has decimal part equals 0.0 the number can be considered even or odd depending on its integer part.
Sample Input:
2

Sample Output:
True


Sample Input:
-4

Sample Output:
True



Memory limit: 256 MB
Time limit: 5 seconds
