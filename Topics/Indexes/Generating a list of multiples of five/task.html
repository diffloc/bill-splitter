Generating a list of multiples of five

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
You are given an integer n. Your goal is to output a list of n integers that satisfy the following conditions: the ith integer of the list is equal to its 1-based position in the list multiplied by 5. Ensure your 0-indexed list starts with 5. For example, if n equals 3, your program should return the list [5, 10, 15] because the 1st, 2nd, and 3rd elements are 5*1, 5*2, and 5*3 respectively.
Sample Input:
1

Sample Output:
[5]


Sample Input:
3

Sample Output:
[5, 10, 15]



Memory limit: 256 MB
Time limit: 5 seconds
