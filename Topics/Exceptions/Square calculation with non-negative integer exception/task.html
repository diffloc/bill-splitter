Square calculation with non-negative integer exception

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Given an integer input n, write a script that will calculate the square of n and print it out. However, if n is less than 0, your script should raise an exception with the message: 'Input must not be less than 0'.
Sample Input:
5

Sample Output:
25


Sample Input:
-5

Sample Output:
Input must not be less than 0



Memory limit: 256 MB
Time limit: 5 seconds
