Find all unique three-digit combinations from a string

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Given a string of digits, write a program that finds all the possible three-digit combinations from the string where each digit occurs exactly once in each combination. The combinations should be printed in ascending order without duplicates. All numbers are non-negative integers.
Sample Input:
123

Sample Output:
123
132
213
231
312
321


Sample Input:
112

Sample Output:
112
121
211



Memory limit: 256 MB
Time limit: 5 seconds
